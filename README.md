# Json Login Api

A new iOS project.

## Getting Started

This project is a starting point for a iOS application.

This application contains the following knowledge:

- iOS 15

- User defaults

- Api
  
- MVVM
  
- Predicates

- LinearGradient

- DispatchQueue
